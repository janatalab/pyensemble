Prolific Integration

Prolific is a platform for recruiting participants for online studies. This page provides tools for integrating PyEnsemble with Prolific.

Create Multi-Day Study
Execute the following steps in order to create a multi-experiment PyEnsemble study and associated Prolific studies, register a test subject, and publish the multi-experiment PyEnsemble study to Prolific that the test subject can then participate in.

  1. Create a multi-experiment PyEnsemble study and associated Prolific studies
  2. Register a test subject with a Prolific participant ID
  3. Register a Prolific test subject with an email account
  4. Transition the multi-experiment PyEnsemble study to test mode
  5. Publish the multi-experiment PyEnsemble study to Prolific
Utility functions
These are various utility functions.
  • Delete Prolific studies - Delete all Prolific studies associated with the current PyEnsemble study.
  • Delete PyEnsemble study - Delete the current PyEnsemble study, experiments, forms, and questions.
