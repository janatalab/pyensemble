BEWARE IF UPLOADING A .CSV FILE EXPORTED FROM MICROSOFT EXCEL!

Microsoft Excel might preprend hidden junk characters to the first column name when saving out to a .csv file, which will prevent the information from the column being written to the stimulus table. After exporting from Excel to a .csv file, open the .csv file in a basic text editor and save it out again to get rid of the hidden characters.
